In Sync is a Next.js app for connecting what you trained with what Oura detected.
You can log structured gym workouts (exercises, sets, reps, optional weight), view Oura workout data by date range, and manually link ring-detected sessions to your logged workouts.

Built as a practical v1: Oura OAuth + API integration, Supabase-backed workout/link storage, and a simple 3-page flow (/workouts, /oura, /link) designed to expand into richer recovery/training insights over time.
